## Key Design Patterns for DR OPS Application

### 1. Dark Mode Foundation (CONFIRMED)
- 90% of references use dark mode as primary/only theme
- Background colors cluster around #0a0a0a to #1a1a1a
- Card surfaces are 1-2 steps lighter than background (#1a-#2a)
- This aligns perfectly with DR OPS dark-mode-only requirement

### 2. Accent Color Strategy
- **Primary accent:** Teal/cyan for interactive elements, links, highlights
- **Success/positive:** Green (#22c55e) for growth, completed states, positive metrics
- **Warning/negative:** Red/coral for losses, alerts, errors
- **Premium/brand:** Purple/violet for brand differentiation
- **Recommendation for DR OPS:** Teal/cyan primary + green positive + red negative + purple for spy/premium features

### 3. Data-Dense Dashboard Patterns
- Cards as primary data containers, each with embedded micro-visualization
