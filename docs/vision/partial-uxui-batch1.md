## Executive Summary - Dominant Patterns

### 1. Color & Theme
- **Dark mode is overwhelmingly dominant** -- 30 of 32 images use dark backgrounds (only 2 light-themed)
- **Primary backgrounds:** Pure black (#000000), near-black (#0A0A0A, #0D0D0D), dark navy (#0B1120, #0D1117, #0F172A)
- **Card backgrounds:** Elevated dark gray (#141414, #1A1A1A, #1F1F1F, #1E1E2E)
- **Accent colors recur consistently:**
  - Purple/Violet: #7C3AED, #8B5CF6, #A855F7 (most common accent)
  - Green/Emerald: #10B981, #22C55E, #84CC16
  - Teal/Cyan: #06B6D4, #14B8A6, #2DD4BF
  - Blue: #3B82F6, #3B82F6
  - Orange/Amber: #F97316, #EA580C, #EAB308
  - Red/Rose: #EF4444, #F43F5E
- **Text colors:** White (#FFFFFF) primary, muted gray (#949494, #6B7280) secondary

### 2. Layout Patterns
- **Sidebar + Content** is the dominant dashboard pattern (8+ images)
- **Card grid layouts** for dashboards with multiple metric widgets
- **3-column pricing cards** (4 images -- very common pattern)
- **Full-width scrolling** for landing pages
- **Node/canvas layouts** for workflow builders (3 images)
- **Centered modals/components** for focused interactions

### 3. Design Style
- **Glassmorphism** is the defining style -- frosted glass cards with blur, subtle borders, transparency
- **Subtle glow effects** -- ambient colored light behind cards (purple, blue, green)
- **Border-based card separation** -- thin borders (#1F1F1F to #2D2D2D) rather than shadows
- **No neumorphism** -- flat with depth achieved through layering and transparency
- **Rounded corners everywhere** -- radius 8-16px standard

### 4. Typography
- **Sans-serif dominant** -- clean geometric sans-serifs throughout
- **Large headline numbers** for financial/metric display (scores, balances, prices)
- **One serif heading** observed (pricing page "Choose your Plan")
- **Weight contrast** -- bold headings (600-700), regular body (400), light secondary (300)

### 5. UI Components
- **Charts:** Line charts, bar charts, donut/pie charts, sparklines, area charts
- **Progress indicators:** Linear progress bars, circular percentage rings
- **Cards:** Metric cards, pricing cards, status cards, profile cards
- **Navigation:** Collapsible sidebars, dropdown menus, tab bars
- **Forms:** Dark-bordered inputs, filter panels, search bars
- **Buttons:** Filled CTAs (accent color), ghost/outline secondary, glassmorphic tertiary
- **Avatars:** Circular with status indicators

### 6. Mood/Atmosphere
- **Professional-futuristic** -- tech-forward but usable
- **Premium/luxury feel** through darkness and subtle lighting
- **Data-dense but organized** -- multiple widgets without clutter
- **Fintech/DeFi aesthetic** heavily represented (crypto, banking, trading)

### Component Library Needs

Based on this analysis, the DR OPS system would benefit from:

1. **Glassmorphism Card Component** -- frosted glass with configurable blur, border, and background opacity
2. **Ambient Glow Wrapper** -- configurable color glow behind elevated components
3. **Data Metric Card** -- large number + label + trend indicator + optional sparkline
4. **Sidebar Navigation** -- collapsible, with icon-only mode, section grouping, active state with accent color
5. **Progress Indicators** -- linear bars (multi-color) and circular rings with percentage
6. **Chart Components** -- line, bar, donut with dark theme colors pre-configured
7. **Tab Navigation** -- pill-style and underline variants
8. **Modal/Dialog** -- glassmorphism style with optional ambient glow
9. **Form Inputs** -- dark-bordered, rounded, with placeholder styling
10. **Filter Panel** -- tab-based with badge counts and range inputs
